# Tutorial: Use Azure Cache for Redis as a Semantic Cache

## Background
This repo contains a [juypter notebook](https://jupyter.org/) file with the code used in the [Use Azure Cache for Redis as a semantic cache](https://aka.ms/semanticcachetutorial) tutorial located in the Azure Cache for Redis documentation. Use this example to follow along with the tutorial or as a basis for your own learning.
